# Forecastly

Forecastly is a simple and beginner-friendly weather app that helps users check real-time weather details like temperature, humidity, and wind speed.  
This project is built using **HTML, CSS, and JavaScript**.

---

## 🚀 Features
- Search weather by city name
- Shows temperature, humidity, and wind speed
- Clean and simple user interface
- Beginner-friendly project
- Responsive design

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 🎯 Purpose of the Project
I created this project to practice my frontend development skills and learn how to build a real-world web application.  
This project helped me understand how to design a user interface, take user input, and display weather information in a simple way.
